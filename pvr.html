Pointer Vs Reference

pointer

A Pointer is a Variable in C++
It's a Variable has address for memory
You have to allocate memory for Pointer
You can access to memory as read that address

Reference

It's an alias for existig memory
You don't have to allocate memory
A Variable has another name!
